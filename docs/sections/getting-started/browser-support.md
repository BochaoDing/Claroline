# Browser support

## Recommended

| Browser                  | Version 
| ------------------------ | ------- 
| Mozilla Firefox          | latest  
| Chromium                 | latest  


## Supported

| Browser                      | Version 
| ---------------------------- | ------- 
| Mozilla Firefox              | latest  
| Chromium                     | latest
| Google Chrome                | latest
| Microsoft Internet Explorer  | >= 11  
| Microsoft Internet Edge      | latest
| Safari                       | latest


> For the complete list of compatible browsers visit [CanIUse](http://caniuse.com/#feat=mutationobserver).
